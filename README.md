# Credit-Score-Classification
The project I chose is a Credit Score Classification project. The dataset is from Paisabazaar, which is one of India's largest online financial marketplaces and a platform that helps people compare and apply for loans, credit cards, and other financial products

#Dataset:
https://drive.google.com/file/d/1P7f1fa0v3hmp5Lc3pbzfVRKOoxY9B66B/view

Number of rows: 1,00,000, Number of columns: 28 

| Feature           | Description     |
| ----------------- | --------------- |
| Age               | Customer age    |
| Income            | Annual income   |
| Outstanding Debt  | Debt amount     |
| Credit History    | Credit length   |
| Payment Behaviour | Payment pattern |
